##Objetivo del proyecto
- gestionar de manera eficiente la información de los tour disponibles de la agencia Llanquihue Tours clasificados por tipo, para mostrarlos en catálogos según las preferencias de sus clientes.
- 
##Estructura del proyecto.
- A continuación se muestra una imagen general de la estructura del proyecto:
```md
📁 src/
├──ui/          # Clase principal con el método main
├──model/       # Clases de dominio (Tour)
├──data/        # Utilidades (Gestor Datos)
└──resorces/    # Recursos (Lista de Tours.txt)
```

Instrucciones para ejecutar el proyecto

1. Clona el repositorio desde GitHub:
https://github.com/Cris-faena/LlanquihueTourApp.git
2. Abre el proyecto en Intelli J IDEA.
3. Ejecuta el archivo Main.Java. desde el paquete "App".
4. Sigue las instrucciones en consola o la interfaz gráfica.
